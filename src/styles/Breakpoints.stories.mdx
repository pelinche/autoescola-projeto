# Breakpoints

Breakpoints for responsive layouts:

<table>
  <tr>
    <th>Extra Small(xs)</th>
    <td>Default</td>
  </tr>
  <tr>
    <th>Small (sm)</th>
    <td>>={BreakpointSizes.sm}</td>
  </tr>
  <tr>
    <th>Medium</th>
    <td>>={BreakpointSizes.md}</td>
  </tr>
  <tr>
    <th>Large</th>
    <td>>={BreakpointSizes.lg}</td>
  </tr>
  <tr>
    <th>Super Large</th>
    <td>>={BreakpointSizes.xl}</td>
  </tr>
</table>

Using in styled component:

```js
impor { breakAt, BreakpointSizes} from Breakpoints;

const Title = styled.h1`
  font-size: 1rem;
  
  ${breakAt(BreakpointSizes.md)}{
    font-size: 2rem;
  }
  `;
```
